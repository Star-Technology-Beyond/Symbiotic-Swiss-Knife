# Tool Adding & Ejecting

The Symbiotic Swiss Knife can hold one of each tool type at a time. Tools can be added
and removed at any point without destroying the multitool.

## Adding a Tool

To add a tool, place the multitool and the GregTech tool you want to install together in
a crafting grid. The tool will be absorbed into the multitool and become available as a
mode.

A tool can only be installed if that tool type is not already present in the multitool. 
The tool must also be of full durability to be inserted. Repair the tool first!

For example, you cannot install a second pickaxe, but you can install both a pickaxe and
a Mining Hammer since they are different tool types.

## Ejecting a Tool

Open the radial selector with <KeyBind id="key.symbiotic_swiss_knife.multitool_selector" />
and **Shift-click** the tool you want to remove. The tool will be ejected from the
multitool and added to your inventory, or dropped at your feet if your inventory is full.

## Warning

Multitool ejection does not preserve everything about the tool you originally inserted.

Mainly these things will be lost:
- Enchantments
- Inserted Charge
