# Getting Started

The Symbiotic Swiss Knife is a single multitool that can hold all your GregTech tools
at once, switching between them in one slot.

## Crafting the Multitool

The empty multitool is crafted like a standard GregTech electric tool. It starts with no
tools installed and cannot do anything until at least one tool is added.
